Ang RKAligner ay isang mekanismo na awtomatikong naghahanay ng mga katumbas na pangungusap ng mga wikang Cebuano at Filipino na kung saan nilalayon nitong bumuo ng parallel corpus sa nasabing mga wika para sa machine translation. Pagkatapos i-upload ang mga source at target .csv file ng mga wikang Cebuano at Filipino, awtomatiko itong magpapakita ng paghahanay ng mga katumbas na pangungusap at ang kaukulang iskor nito. Ang RKAligner ay hindi lamang naglalayong maghanay ng mga pangungusap ngunit maaari rin itong magbigay ng mga pagsasalin kung ang source file lamang ang tanging magagamit.

Dapat nating isaalaala na tumatanggap lamang ang RKAligner ng .csv files. Maaari mo lamang ma download ang file naglalaman ng nakahanay na mga pangugusap kapag ito ay nasuri na ng aming mga eksperto sa wika o linggwista.
